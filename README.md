# Description

Игра-головоломка  типа "puzzle -tag".
В качестве награды игрок получает новый факт о белках или насмешку.
Локализация на : Русский, Английский. Смена языка как функция для игрока.

# Techologies

Язык Java/Kotlin
Работа с базовыми элементами:
- XML;
- Image;
- Animations;
- Styles;
- Fragments;
- Intents;
- Navigations;

- Arrays;
- Collections;
- Interfaceces;

Сохранение данных:
- SharedPreferences;
- saveInstanceState;
