# MyTag
MyTag v1.1.1

Игра-головоломка  типа "puzzle -tag".
В качестве награды игрок получает интересные факты о белках.

Язык Java/Kotlin
Работа с базовыми элементами Android и Java/Kotlin:
- XML;
- Image;
- Fragments;
- Styles;
- Intents;
- Navigations;

* Collections;
* Arrays;
* Strings;
* Interfaceces; 

Сохранение данных:
- SharedPreferences;
- saveInstanceState;
